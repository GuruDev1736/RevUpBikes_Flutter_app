# Banner Images Directory

This directory contains banner images for the promotional sliders in the RevUp bike rental app.

## Banner Images Specifications:
- **Dimensions**: 800x400px (16:8 aspect ratio)
- **Format**: JPG or PNG
- **File Size**: Preferably under 300KB for optimal loading
- **Quality**: High resolution for crisp display

## Banner Types:
1. **explore_city.jpg** - City exploration and urban cycling
2. **electric_bike.jpg** - Electric bikes and eco-friendly transportation
3. **adventure.jpg** - Adventure and mountain biking experiences
4. **mountain_biking.jpg** - Extreme mountain biking and trails
5. **city_cycling.jpg** - Urban cycling and commuting
6. **special_offers.jpg** - Promotional offers and discounts
7. **weekend_rides.jpg** - Weekend cycling activities
8. **eco_friendly.jpg** - Environmental and sustainable transportation

## Network Images:
The app currently uses high-quality images from Unsplash API for banners. Local images in this directory serve as offline fallbacks.

## Usage:
- Network images load first for the best visual experience
- Local images are used when network is unavailable
- Gradient backgrounds serve as the final fallback
